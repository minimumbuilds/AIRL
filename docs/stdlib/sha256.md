# sha256

## u32
**Signature:** `(x : i64) -> i64`
**Intent:** Mask value to 32 bits

---

## u32-rotr
**Signature:** `(x : i64) (n : i64) -> i64`
**Intent:** 32-bit right rotate

---

## u32-not
**Signature:** `(x : i64) -> i64`
**Intent:** 32-bit bitwise NOT

---

## u32-add2
**Signature:** `(a : i64) (b : i64) -> i64`
**Intent:** Add two values mod 2^32

---

## u32-add5
**Signature:** `(a : i64) (b : i64) (c : i64) (d : i64) (e : i64) -> i64`
**Intent:** Add five values mod 2^32

---

## list-nth
**Signature:** `(lst : List) (n : i64) -> i64`
**Intent:** Get nth element of a list (0-indexed)

---

## sha256-ch
**Signature:** `(x : i64) (y : i64) (z : i64) -> i64`
**Intent:** Ch(x,y,z) = (x AND y) XOR (NOT x AND z)

---

## sha256-maj
**Signature:** `(x : i64) (y : i64) (z : i64) -> i64`
**Intent:** Maj(x,y,z) = (x AND y) XOR (x AND z) XOR (y AND z)

---

## sha256-bsig0
**Signature:** `(x : i64) -> i64`
**Intent:** Big Sigma 0: ROTR2 XOR ROTR13 XOR ROTR22

---

## sha256-bsig1
**Signature:** `(x : i64) -> i64`
**Intent:** Big Sigma 1: ROTR6 XOR ROTR11 XOR ROTR25

---

## sha256-ssig0
**Signature:** `(x : i64) -> i64`
**Intent:** Small sigma 0: ROTR7 XOR ROTR18 XOR SHR3

---

## sha256-ssig1
**Signature:** `(x : i64) -> i64`
**Intent:** Small sigma 1: ROTR17 XOR ROTR19 XOR SHR10

---

## hex-nibble
**Signature:** `(n : i64) -> i64`
**Intent:** Convert 0-15 to ASCII hex character byte value

---

## hex-byte-bytes
**Signature:** `(b : i64) -> Bytes`
**Intent:** Convert byte value to 2-character hex Bytes

---

## bytes-to-hex-loop
**Signature:** `(buf : Bytes) (i : i64) (len : i64) -> List`
**Intent:** Convert each byte to hex, returning list of 2-byte Bytes

---

## be32-get
**Signature:** `(buf : Bytes) (off : i64) -> i64`
**Intent:** Read big-endian 32-bit word from Bytes at byte offset

---

## be32-to-list
**Signature:** `(w : i64) -> List`
**Intent:** Convert 32-bit word to list of 4 big-endian byte Bytes

---

## be64-to-list
**Signature:** `(val : i64) -> List`
**Intent:** Convert 64-bit value to list of 8 big-endian byte Bytes

---

## make-zero-bytes
**Signature:** `(n : i64) -> List`
**Intent:** Create list of n zero-byte Bytes values

---

## sha256-pad
**Signature:** `(msg : Bytes) -> Bytes`
**Intent:** Pad message per FIPS 180-4 section 5.1.1

---

## sha256-schedule-init
**Signature:** `(block : Bytes) (base : i64) (i : i64) -> List`
**Intent:** Extract 16 big-endian 32-bit words from a 64-byte block

---

## sha256-schedule-extend
**Signature:** `(window : List) (result-rev : List) (i : i64) -> List`
**Intent:** Build W[16..63] extension words using a 16-word sliding window

---

## sha256-schedule
**Signature:** `(block : Bytes) (base : i64) -> List`
**Intent:** Build full 64-word message schedule from a 64-byte block

---

## sha256-compress-round
**Signature:** `(state : List) (ki : i64) (wi : i64) -> List`
**Intent:** One SHA-256 compression round updating 8 working variables

---

## sha256-compress-loop
**Signature:** `(state : List) (k : List) (w : List) (i : i64) -> List`
**Intent:** Run 64 compression rounds using head/tail for K and W

---

## sha256-add-hash
**Signature:** `(h : List) (state : List) -> List`
**Intent:** Add compressed state back to current hash values

---

## sha256-compress-block
**Signature:** `(hash : List) (k : List) (padded : Bytes) (base : i64) -> List`
**Intent:** Compress one 64-byte block against current hash state

---

## sha256-process-blocks
**Signature:** `(hash : List) (k : List) (padded : Bytes) (off : i64) (len : i64) -> List`
**Intent:** Process all 64-byte blocks of padded message

---

## sha256-hash-to-bytes
**Signature:** `(hash : List) -> Bytes`
**Intent:** Convert 8 hash words to 32-byte digest

---
