# math

## abs
**Signature:** `(x : i64) -> i64`
**Intent:** Return the absolute value of x

---

## min
**Signature:** `(a : i64) (b : i64) -> i64`
**Intent:** Return the smaller of two values

---

## max
**Signature:** `(a : i64) (b : i64) -> i64`
**Intent:** Return the larger of two values

---

## clamp
**Signature:** `(x : i64) (lo : i64) (hi : i64) -> i64`
**Intent:** Clamp x to the range [lo, hi]

---

## sign
**Signature:** `(x : i64) -> i64`
**Intent:** Return -1, 0, or 1 for negative, zero, or positive

---

## even?
**Signature:** `(x : i64) -> bool`
**Intent:** Check if x is even

---

## odd?
**Signature:** `(x : i64) -> bool`
**Intent:** Check if x is odd

---

## pow
**Signature:** `(base : i64) (exp : i64) -> i64`
**Intent:** Integer exponentiation using fast squaring

---

## gcd
**Signature:** `(a : i64) (b : i64) -> i64`
**Intent:** Compute greatest common divisor using Euclidean algorithm

---

## lcm
**Signature:** `(a : i64) (b : i64) -> i64`
**Intent:** Compute least common multiple

---

## sum-list
**Signature:** `(xs : List) -> i64`
**Intent:** Sum all elements in a list

---

## product-list
**Signature:** `(xs : List) -> i64`
**Intent:** Multiply all elements in a list

---
