Write a function called `permutations` that generates all permutations of a list.

Requirements:
- Return a list of all possible orderings of the input elements
- The number of permutations should be N! for a list of length N
- An empty list has one permutation: the empty list
- A single-element list has one permutation: itself
- Order of the permutations doesn't matter

Test case: permutations([1]) should return [[1]]
Test case: permutations([1, 2]) should return [[1, 2], [2, 1]]

Print the result of calling the function with argument [1, 2, 3].
