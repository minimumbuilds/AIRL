Write a function called `list-intersection` that takes two lists and returns a list of elements present in both.

Requirements:
- The result should contain no duplicates
- Preserve the order of first occurrence from the first list
- An empty list intersected with any list returns an empty list

Test case: list-intersection([1, 2, 3, 4], [3, 4, 5, 6]) should return [3, 4]
Test case: list-intersection([1, 2], [3, 4]) should return []

Print the result of calling the function with arguments [1, 2, 3, 4, 5] and [2, 4, 6, 8].
