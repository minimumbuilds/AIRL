Write a function called `running-sum` that takes a list of integers and returns a list of cumulative sums.

Requirements:
- The result must have the same length as the input
- Each element at index i is the sum of input elements 0 through i
- An empty list returns an empty list

Test case: running-sum([1, 2, 3, 4]) should return [1, 3, 6, 10]
Test case: running-sum([]) should return []

Print the result of calling the function with argument [1, 2, 3, 4, 5].
