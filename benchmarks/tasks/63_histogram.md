Write a function called `histogram` that takes a list of integers (each between 0 and 9) and returns a string showing a horizontal bar chart.

Requirements:
- Each line shows the digit, a colon, a space, and '#' repeated once per occurrence
- Only show digits that appear at least once
- Sort lines by digit ascending
- Each line ends with a newline

Test case: histogram([1, 2, 2, 3, 3, 3]) should return "1: #\n2: ##\n3: ###\n"

Print the result of calling the function with argument [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5].
