Write a function called `has-all-vowels` that checks whether a string contains all five vowels (a, e, i, o, u).

Requirements:
- Check should be case-insensitive
- The vowels do not need to be in order or adjacent
- Return true if all five vowels are present, false otherwise
- An empty string returns false

Test case: has-all-vowels("education") should return true
Test case: has-all-vowels("hello") should return false

Print the result of calling the function with argument "sequoia".
