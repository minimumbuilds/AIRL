Write a function called `is-palindrome` that takes a string and returns true if it reads the same forwards and backwards, false otherwise.

Requirements:
- The check should be case-sensitive
- An empty string is considered a palindrome
- Single-character strings are palindromes

Print the result of calling the function with argument "racecar".

---TESTS---
is-palindrome("racecar") => true
is-palindrome("hello") => false
is-palindrome("") => true
is-palindrome("a") => true
