Write a function called `caesar-cipher` that takes a string and an integer shift, and returns the string with each letter shifted by that many positions in the alphabet.

Requirements:
- The shift must be between 0 and 25 inclusive
- Non-letter characters are unchanged
- The result must have the same length as the input

Test case: caesar-cipher("abc", 3) should return "def"
Test case: caesar-cipher("xyz", 3) should return "abc"

Print the result of calling the function with arguments "hello" and 3.
