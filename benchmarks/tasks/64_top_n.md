Write a function called `top-n` that takes a list of integers and an integer N, and returns the N largest elements in descending order.

Requirements:
- Sort the list in descending order
- Return the first N elements
- If N is greater than the list length, return the entire sorted list
- Duplicates are preserved

Test case: top-n([3, 1, 4, 1, 5, 9, 2, 6], 3) should return [9, 6, 5]
Test case: top-n([10, 20], 5) should return [20, 10]

Print the result of calling the function with arguments [7, 2, 9, 4, 1, 8, 3] and 4.
