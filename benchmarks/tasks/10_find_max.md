Write a function called `find-max` that takes a non-empty list of integers and returns the maximum element.

Requirements:
- The input list must not be empty
- The result must be greater than or equal to every element in the list
- The result must be an element that exists in the list

Test case: find-max([3, 1, 4, 1, 5, 9]) should return 9
Test case: find-max([42]) should return 42

Print the result of calling the function with argument [3, 7, 2, 8, 1, 9, 4].
