Write a function called `merge-sorted` that takes two sorted lists of integers and merges them into a single sorted list.

Requirements:
- Both input lists must already be sorted in ascending order
- The result must be sorted in ascending order
- The result length must equal the sum of the input lengths

Test case: merge-sorted([1, 3, 5], [2, 4, 6]) should return [1, 2, 3, 4, 5, 6]
Test case: merge-sorted([], [1, 2]) should return [1, 2]

Print the result of calling the function with arguments [1, 3, 5, 7] and [2, 4, 6, 8].
