Write a function called `absolute-value` that takes an integer `x` and returns its absolute value.

Requirements:
- If `x` is negative, return `-x`
- If `x` is zero or positive, return `x`
- The function must ensure the result is always >= 0

Test case: absolute-value(-5) should return 5
Test case: absolute-value(3) should return 3
Test case: absolute-value(0) should return 0

Print the result of calling the function with argument -42.
