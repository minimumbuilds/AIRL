Write a function called `is-all-upper` that checks whether every letter in a string is uppercase.

Requirements:
- Non-letter characters (spaces, digits, punctuation) are ignored
- An empty string returns true
- A string with no letters returns true
- Compare the string to its uppercased version

Test case: is-all-upper("HELLO") should return true
Test case: is-all-upper("Hello") should return false
Test case: is-all-upper("ABC 123") should return true

Print the result of calling the function with argument "HELLO WORLD".
