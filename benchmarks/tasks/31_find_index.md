Write a function called `find-index` that takes a list and a value, returning the index of the first occurrence of that value, or -1 if not found.

Requirements:
- Return the 0-based index of the first match
- Return -1 if the value is not in the list
- An empty list returns -1
- Use equality comparison for matching

Test case: find-index([10, 20, 30, 40], 30) should return 2
Test case: find-index([10, 20, 30], 99) should return -1

Print the result of calling the function with arguments [5, 10, 15, 20, 25] and 15.
