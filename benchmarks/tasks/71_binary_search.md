Write a function called `binary-search` that takes a sorted list of integers and a target value, returning the index of the target or -1 if not found.

Requirements:
- The input list must be sorted in ascending order
- Use recursive binary search (not linear scan)
- Return the 0-based index of the target
- Return -1 if the target is not in the list

Test case: binary-search([1, 3, 5, 7, 9, 11], 7) should return 3
Test case: binary-search([1, 3, 5, 7, 9], 4) should return -1

Print the result of calling the function with arguments [2, 4, 6, 8, 10, 12, 14, 16] and 10.
