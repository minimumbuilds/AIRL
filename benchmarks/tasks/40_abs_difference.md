Write a function called `abs-difference` that takes two integers and returns the absolute value of their difference.

Requirements:
- The result must always be non-negative
- abs-difference(a, b) should equal abs-difference(b, a)

Test case: abs-difference(10, 3) should return 7
Test case: abs-difference(3, 10) should return 7
Test case: abs-difference(5, 5) should return 0

Print the result of calling the function with arguments 42 and 17.
