Write a function called `second-largest` that takes a list of at least two distinct integers and returns the second-largest element.

Requirements:
- The list has at least two elements
- Return the element that is smaller than exactly one other element
- If the two largest values are equal, return that value

Test case: second-largest([3, 1, 4, 1, 5, 9]) should return 5
Test case: second-largest([10, 20]) should return 10

Print the result of calling the function with argument [7, 2, 9, 4, 1, 8].
