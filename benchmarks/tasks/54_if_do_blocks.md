Write a function called `fizzbuzz-list` that takes an integer N and returns a list of strings for numbers 1 through N.

Requirements:
- If a number is divisible by both 3 and 5, use "FizzBuzz"
- If divisible by 3 only, use "Fizz"
- If divisible by 5 only, use "Buzz"
- Otherwise, use the number as a string
- Return a list of strings

Test case: fizzbuzz-list(5) should return ["1", "2", "Fizz", "4", "Buzz"]
Test case: fizzbuzz-list(15) should end with "FizzBuzz"

Print the result of calling the function with argument 15.
