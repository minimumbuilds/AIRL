Write a function called `to-binary` that converts a non-negative integer to its binary string representation.

Requirements:
- 0 should return "0"
- Use repeated division by 2 and collect remainders
- No leading zeros (except for input 0)
- Input must be non-negative

Test case: to-binary(0) should return "0"
Test case: to-binary(10) should return "1010"
Test case: to-binary(255) should return "11111111"

Print the result of calling the function with argument 42.
