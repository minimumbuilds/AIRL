Write a function called `is-anagram` that checks whether two strings are anagrams of each other (contain the same characters in any order).

Requirements:
- Case-insensitive comparison
- Ignore spaces
- Two empty strings are anagrams of each other
- Characters must appear the same number of times in both strings

Test case: is-anagram("listen", "silent") should return true
Test case: is-anagram("hello", "world") should return false
Test case: is-anagram("Astronomer", "Moon starer") should return true

Print the result of calling the function with arguments "triangle" and "integral".
