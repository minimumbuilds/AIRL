Write a function called `describe-shape` that takes a shape variant and returns a string description.

Requirements:
- Define shapes using match on variant tags: (Circle radius), (Rectangle width height), (Triangle base height)
- Circle description: "circle with radius N"
- Rectangle description: "rectangle WxH"
- Triangle description: "triangle base B height H"
- Use match to handle each variant

Create a Circle with radius 5, a Rectangle with width 3 and height 4, and a Triangle with base 6 and height 2. Print the description of each.

Print the result of calling describe-shape with a Rectangle of width 10 and height 5.
