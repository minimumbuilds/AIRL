Write a function called `flatten-sort-unique` that takes a list of lists of integers, flattens it, removes duplicates, and sorts the result in ascending order.

Requirements:
- Flatten one level of nesting
- Remove duplicate values
- Sort the result in ascending order
- An empty outer list returns an empty list

Test case: flatten-sort-unique([[3, 1], [4, 1], [5, 9, 2]]) should return [1, 2, 3, 4, 5, 9]

Print the result of calling the function with argument [[5, 3, 1], [2, 3, 4], [1, 6, 5]].
