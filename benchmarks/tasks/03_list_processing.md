Write a function called `sum-doubled-evens` that takes a list of integers and returns the sum of all even numbers doubled.

Requirements:
- Filter the list to keep only even numbers
- Double each remaining number
- Sum the results
- An empty list should return 0
- The result must be non-negative if all inputs are non-negative

Test case: sum-doubled-evens([1, 2, 3, 4, 5, 6]) should return 24 (2*2 + 4*2 + 6*2 = 24)

Print the result of calling the function with the list [1, 2, 3, 4, 5, 6].
