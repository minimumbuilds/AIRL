Write a function called `count-vowels` that takes a string and returns the number of vowels (a, e, i, o, u, case-insensitive).

Requirements:
- Count both uppercase and lowercase vowels
- The result must be >= 0
- The result must be <= the length of the string

Test case: count-vowels("hello world") should return 3
Test case: count-vowels("xyz") should return 0

Print the result of calling the function with argument "hello world".
