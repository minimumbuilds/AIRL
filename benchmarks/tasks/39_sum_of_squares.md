Write a function called `sum-of-squares` that takes a list of integers and returns the sum of each element squared.

Requirements:
- Square each element, then sum the results
- An empty list returns 0
- Negative numbers squared become positive

Test case: sum-of-squares([1, 2, 3]) should return 14 (1 + 4 + 9)
Test case: sum-of-squares([-2, 3]) should return 13 (4 + 9)
Test case: sum-of-squares([]) should return 0

Print the result of calling the function with argument [3, -1, 4, -1, 5].
