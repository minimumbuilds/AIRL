Write a function called `remove-duplicates` that takes a list of integers and returns a new list with duplicates removed, preserving the first occurrence order.

Requirements:
- The result length must be <= the input length
- Each element in the result must appear exactly once
- The relative order of first occurrences is preserved

Test case: remove-duplicates([1, 2, 3, 2, 1, 4]) should return [1, 2, 3, 4]
Test case: remove-duplicates([]) should return []

Print the result of calling the function with argument [5, 3, 5, 1, 3, 2, 1].
