Write a function called `reverse-list` that takes a list of integers and returns a new list with elements in reverse order.

Requirements:
- The result must have the same length as the input
- An empty list reversed is an empty list
- The function must not modify the original list

Test case: reverse-list([1, 2, 3, 4]) should return [4, 3, 2, 1]
Test case: reverse-list([]) should return []

Print the result of calling the function with argument [1, 2, 3, 4, 5].
