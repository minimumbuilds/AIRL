Write a function called `power` that computes integer exponentiation: base raised to the exponent.

Requirements:
- The exponent must be >= 0
- When base > 0, the result must be >= 1
- power(x, 0) should return 1 for any x

Test case: power(2, 10) should return 1024
Test case: power(3, 0) should return 1

Print the result of calling the function with arguments 2 and 10.
