Write a function called `is-prime` that checks whether an integer is a prime number.

Requirements:
- Numbers less than 2 are not prime
- 2 is prime
- Check divisibility from 2 up to the square root of n
- Return true if the number has no divisors other than 1 and itself

Test case: is-prime(7) should return true
Test case: is-prime(4) should return false
Test case: is-prime(1) should return false
Test case: is-prime(2) should return true

Print the result of calling the function with argument 29.
