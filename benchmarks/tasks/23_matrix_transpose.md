Write a function called `transpose` that takes a 2D list (list of lists) and returns its transpose.

Requirements:
- Rows become columns and columns become rows
- All inner lists must have the same length
- transpose of an m x n matrix is an n x m matrix

Test case: transpose([[1, 2, 3], [4, 5, 6]]) should return [[1, 4], [2, 5], [3, 6]]

Print the result of calling the function with argument [[1, 2], [3, 4], [5, 6]].
