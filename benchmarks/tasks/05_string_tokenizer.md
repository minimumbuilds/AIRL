Write a function called `tokenize` that splits a string into non-empty tokens by whitespace.

Requirements:
- Split the input string by spaces
- Remove any empty strings from the result
- Return the list of tokens
- An empty string should return an empty list

Test case: tokenize("hello  world  foo") should return ["hello", "world", "foo"]
Test case: tokenize("") should return []

Print the result of calling the function with "hello  world  foo".
