Write a function called `list-difference` that takes two lists and returns elements in the first list that are not in the second.

Requirements:
- Preserve the order from the first list
- Remove duplicates from the result
- An empty first list returns an empty list

Test case: list-difference([1, 2, 3, 4, 5], [2, 4]) should return [1, 3, 5]
Test case: list-difference([1, 2, 3], [1, 2, 3]) should return []

Print the result of calling the function with arguments [1, 2, 3, 4, 5, 6] and [2, 4, 6].
