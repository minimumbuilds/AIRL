Write a function called `divmod` that takes two integers and returns a list of [quotient, remainder].

Requirements:
- The quotient is the result of integer division
- The remainder is the modulo result
- The divisor must not be zero
- Return a two-element list [quotient, remainder]

Test case: divmod(17, 5) should return [3, 2]
Test case: divmod(10, 3) should return [3, 1]
Test case: divmod(6, 6) should return [1, 0]

Print the result of calling the function with arguments 23 and 7.
