Write a function called `kebab-to-title` that converts a kebab-case string to Title Case.

Requirements:
- Split the string by hyphens
- Capitalize the first character of each word (using to-upper on the first char)
- Join the words back with spaces
- An empty string returns an empty string

Test case: kebab-to-title("hello-world") should return "Hello World"
Test case: kebab-to-title("one") should return "One"

Print the result of calling the function with argument "my-cool-function".
