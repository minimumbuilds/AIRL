Write a function called `longest-string` that takes a non-empty list of strings and returns the longest one.

Requirements:
- If multiple strings share the maximum length, return the first one
- The input list must not be empty
- Use fold to compare lengths

Test case: longest-string(["hi", "hello", "hey"]) should return "hello"
Test case: longest-string(["a"]) should return "a"

Print the result of calling the function with argument ["cat", "elephant", "dog", "butterfly"].
